5.4.3. Sintaxis de CSS

  propiedad: valor;

5.4.4. Variables CSS

:root {
  --color-primario: #1e90ff;
}
button { background: var(--color-primario); }

5.4.5. Selectores

  • Tipo: p, Clase: .nota, ID: #encabezado
  • Atributo: input[type="email"]
  • Descendiente: nav a, Hijo: ul > li, Hermano: h2 + p
  • Pseudo-clases: a:hover, :focus; Pseudo-elementos: ::before
