5.2. Estructura básica

<!doctype html>
<html lang="es">
  <head>...</head>
  <body>...</body>
</html>

La estructura mínima incluye la declaración !doctype, idioma, metadatos y el cuerpo.

Elementos semánticos comunes

  • <header>, <footer>
  • <nav>, <main>
  • <section>, <article>, <aside>
