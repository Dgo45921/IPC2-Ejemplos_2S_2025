5.3. Componentes básicos

5.3.1. Elementos

Un elemento HTML se compone de etiqueta de apertura, contenido y etiqueta de cierre.

5.3.2. Atributos

Proporcionan información adicional. Ejemplo: <a href="https://ejemplo.com" target="_blank" rel="noopener">enlace</a>.

También hay atributos globales: id, class, title, lang, data-*, style, hidden, etc.

5.3.3. Párrafos

Los párrafos se marcan con <p>. Puedes usar <strong> y <em> para significado (no solo apariencia).

5.3.4. Estilos

Este párrafo usa estilos en línea solo a modo demostrativo (no recomendado en producción).
