5.1. Introducción a HTML5 y CSS

HTML define la estructura semántica del contenido; CSS define su presentación.

¿Qué es HTML5?

Un lenguaje de marcado con elementos semánticos como <header>, <nav>, <main>, <section>, <article> y <footer>.

¿Qué es CSS?

Hojas de estilo en cascada que permiten separar el contenido de su presentación visual.

Ventajas de separar HTML y CSS

  • Mantenibilidad y reutilización.
  • Rendimiento (caché de estilos).
  • Accesibilidad y consistencia.
