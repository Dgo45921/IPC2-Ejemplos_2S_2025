5.4. CSS

5.4.1. Módulos CSS

CSS está dividido en módulos: Selectors, Cascade & Inheritance, Values & Units, Box Model, Backgrounds & Borders, Text, Flexbox, Grid, Transforms, Transitions, Animations, Media Queries, etc.

5.4.2. Conectar HTML y CSS

Formas de conexión:

  1. Etiqueta <link rel="stylesheet" href="styles.css"> (externo).
  2. Etiqueta <style>...</style> (interno).
  3. Atributo style="..." (en línea; desaconsejado en producción).

5.4.3. Sintaxis de CSS

selector {
  propiedad: valor;
  otra-propiedad: otro-valor;
}

5.4.4. Variables CSS (custom properties)

:root {
  --color-primario: #1e90ff;
}
button {
  background: var(--color-primario);
}

5.4.5. Selectores

  • Tipo: p, Clase: .nota, ID: #encabezado
  • Atributo: input[type="email"]
  • Descendiente: nav a, Hijo: ul > li, Hermano adyacente: h2 + p
  • Pseudo-clases: a:hover, :focus, :first-child; Pseudo-elementos: ::before
